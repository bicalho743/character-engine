## Formato de Post (X/Twitter)

**Post principal:** curiosidade real sobre o tema/produto
- Sem link, sem preço, sem nomear produto diretamente
- Deve provocar reação genuína

**Reply (2 minutos depois):** promoção direta
- Com link afiliado obrigatório
- Tom: "já que tô falando nisso..."
- Preço atual + preço anterior quando disponível

## Pipeline Técnico

```
Produto (link ou Shopee API)
    ↓
Web Search (gpt-4o-search-preview) — busca fato real
    ↓
Escritor (GPT-4o) — gera { main: curiosidade, reply: promoção }
    ↓
Crítico (GPT-4o) — valida qualidade e autenticidade
    ↓
Editor (GPT-4o) — reescreve se necessário
    ↓
Link forçado por código no reply (independente do GPT)
    ↓
Post principal no X → 2min → Reply automático
```

## Pilares de Conteúdo

1. **Curiosidade** — fato interessante, verificável, sem produto
2. **Achado** — promoção real com link afiliado
3. **Experiência** — Gi conta algo que viveu ou testou
4. **Resposta** — reply humano a seguidores
