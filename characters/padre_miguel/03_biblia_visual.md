# SKILL 03 — Bíblia Visual
## Personagem: Padre Miguel

---

## Estilo

**REALISTA HIPER-FOTOGRÁFICO.**  
Equivalente a uma fotografia profissional em câmera de alta resolução.  
Não cartoon, não 3D, não ilustração.

A consistência é cirúrgica: qualquer variação de traço entre gerações é percebida pelo público. A imagem-base é lei.

---

## Aparência física

| Atributo | Descrição |
|---|---|
| Idade | 50–55 anos |
| Cabelo | Grisalho nas têmporas, escuro no topo, levemente despenteado |
| Olhos | Castanhos, expressivos, levemente cansados — olhar que já viu muita coisa |
| Barba | Curta, bem cuidada, com fios grisalhos |
| Roupa | Batina preta simples com colarinho romano branco |
| Expressão de repouso | Serena, levemente séria — nunca sorrindo forçado |
| Cenário padrão | Interior de igreja simples, luz suave de vela ou vitral, fundo levemente desfocado |

## Prompt inicial (geração da imagem-base)

```
[character] hyper-realistic Brazilian Catholic priest, age 50-55,
[face] slightly graying temples, dark hair on top, warm brown eyes,
  expressive gaze, short well-trimmed beard with gray streaks,
  calm and kind expression, slight tiredness around the eyes
[clothing] simple black cassock, white Roman collar
[setting] interior of a simple church, soft candlelight and stained
  glass window in background, warm light
[style] hyper-realistic photography, Canon EOS R5, 85mm portrait lens,
  f/2.0 shallow depth of field, cinematic color grading,
  NOT cartoon, NOT 3D, NOT illustration
[quality] 8k, photorealistic, professional portrait photography
[framing] portrait, chest up, character centered facing camera
[lighting] soft warm directional light, subtle shadows
[negative] cartoon, 3d render, illustration, young face, smooth skin,
  perfect teeth, stock photo smile, watermark
[ratio] 9:16 vertical
```

## Prompt de turnaround sheet (4 vistas)

```
[REFERENCE]
Use the attached image (Padre Miguel canon) as the SOLE identity
  reference. SAME person in every view: 50-55 years old, dark hair
  with graying temples, warm brown eyes, short beard with gray
  streaks, simple black cassock, white Roman collar.
[LAYOUT — character turnaround sheet]
One sheet, SAME character in FOUR full-body views, evenly spaced
  left to right, same scale, same neutral A-pose:
  1. FRONT  2. THREE-QUARTER (45°)  3. SIDE PROFILE  4. BACK
[CONSISTENCY RULES]
Identical height, proportions, hair, beard, cassock in all views.
White collar visible in front/3-4/side views. Only rotation changes.
[BACKGROUND]
Clean seamless dark backdrop. Flat soft lighting.
[STYLE]
Hyper-realistic, full body head-to-feet, character model sheet.
[NEGATIVE]
cartoon, different faces, inconsistent character, different heights,
  cropped feet, missing collar, extra characters, watermark
```

```
[LAYOUT — expression sheet]
One image, clean GRID of 8 head-and-shoulders portraits, 4 columns

[THE 8 EXPRESSIONS]
Row 1: 1) sereno 2) acolhedor/sorriso suave 3) pensativo/reflexivo

[BACKGROUND]
Clean seamless dark backdrop in every panel. Soft warm light.
[STYLE]
Hyper-realistic photography, expression reference sheet.
[NEGATIVE]
cartoon, different faces between panels, inconsistent character,
  missing collar, missing beard, watermark, merged panels
```
